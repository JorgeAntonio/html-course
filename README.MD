#### 🧠 Conceptos clave:

  - Protocolo de comunicación.

- **Frontend vs Backend**
- **Ejemplo visual**: Usar [https://developer.mozilla.org/es/docs/Learn/Getting_started_with_the_web/How_the_Web_works](https://developer.mozilla.org/es/docs/Learn/Getting_started_with_the_web/How_the_Web_works)

### **8:35 a.m. – 9:00 a.m. | Estructura básica de HTML**

#### 🧱 Explicar:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Mi Primera Página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
  </body>
</html>
```

- **`<!DOCTYPE html>`** → define que es HTML5
- **`<html>`**, **`<head>`**, **`<body>`**
- El navegador interpreta esto y lo presenta visualmente.

#### 💡 Tip:

Mostrar en vivo cómo un cambio en el HTML impacta en la web (abrir archivo .html en navegador).

---

### **9:00 a.m. – 9:20 a.m. | Etiquetas básicas**

#### 🎯 Etiquetas:

- **Encabezados:** `<h1>` hasta `<h6>`
- **Párrafos:** `<p>`
- **Salto de línea:** `<br>`
- **Línea horizontal:** `<hr>`

#### 🧪 Demostración:

```html
<h1>Título principal</h1>
<h2>Subtítulo</h2>
<p>Este es un párrafo de texto.</p>
<p>Otro párrafo con un<br />salto de línea.</p>
<hr />
```

- Mostrar cómo se ven en el navegador.
- Explicar diferencias de jerarquía visual entre h1 y h6.

---

### **9:20 a.m. – 9:35 a.m. | Buenas prácticas y semántica**

#### 🧹 Buenas prácticas:

- Cierre correcto de etiquetas
- Uso de sangrías y comentarios
- No abusar de `<br>` ni `<hr>`

#### 💬 Semántica:

- HTML no es solo para ver, también describe el contenido.
- Por qué no usar `<div>` para todo.
- Breve mención de etiquetas semánticas (se usarán más adelante): `<header>`, `<main>`, `<footer>`

### **9:50 a.m. – 10:50 a.m. | 🧪 Ejercicio práctico: “Tu primera página HTML”**

#### 📄 Requisitos del ejercicio:

- Título principal con `<h1>` que diga “Mi primera página web”
- 2 párrafos con tu presentación personal

- Al menos un `<br>` para separar ideas
- Agregar un `<title>` personalizado

💡 _Supervisar y ayudar a los estudiantes. Corregir errores comunes: etiquetas mal cerradas, errores de sintaxis, etc._
