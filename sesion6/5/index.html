5. Ejercicio: Formulario semántico y accesible

Instrucciones

  1. Utiliza etiquetas semánticas para la estructura general.
  2. Incluye al menos: input de texto, email, radio, checkbox, select.
  3. Usa <label> correctamente y atributos de accesibilidad.
  4. Agrega metaetiquetas y títulos adecuados en el <head>.
