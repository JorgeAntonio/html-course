Buenas Prácticas HTML

Las buenas prácticas en HTML son un conjunto de recomendaciones y normas que se deben seguir para crear páginas web de manera eficiente y organizada.

Las etiquetas semánticas son aquellas que describen el significado de los elementos en la página.

El HTML no solo se ve bonito, también debe tener sentido para el navegador, los buscadores y las personas (por ejemplo, con discapacidad visual que usan lectores de pantalla).
